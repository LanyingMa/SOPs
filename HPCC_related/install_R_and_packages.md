# Use Anaconda   

## install R from anaconda   
```
conda install -c r r
```

## install essential packages that are available in anacoda  
```
conda install -c r r-essentials
```

## instal phyloseq from bioconductor  
```
source("http://bioconductor.org/biocLite.R")
biocLite("RJSONIO")
biocLite("phyloseq")
```
